Star.P.G.
---------
A clasic side scrolling shooter, with added RPG elements.
www.physhexgames.co.nr
paul.wortmann@gmail.com / physhex@gmail.com


Keys:
-----

"Left"    ->    Move starship left.
"Right"   ->    Move starship right.
"Up"      ->    Move starship up.
"Down"    ->    Move starship down.
"Space"   ->    Fire weapon.
"P"       ->    Pause / Resume.
"ESC"	  ->    Menu, Quit.
